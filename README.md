## Overview
This project analyzes device insurance attach percentage performance across stores, branches, and months for Jumbo & Company.  
The goal is to understand historical trends, identify performance patterns, categorize stores, and predict attach percentage for January.

## Dataset
- Store-level attach percentage data
- Time period: August to December
- Metrics: Monthly attach percentages per store and branch

## Analysis Covered
1. Data overview and validation  
2. Month-wise attach percentage trend analysis  
3. Store-wise performance analysis  
4. Branch-wise performance analysis  
5. Store categorization based on performance  
6. January attach percentage prediction  
7. Business insights and recommendations  

## Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

## Output
- Jupyter Notebook with analysis and visualizations  
- Final PDF report summarizing insights and conclusions
